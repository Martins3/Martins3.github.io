| 课程         | 研究内容                                       |
| ------       | -----------------------                        |
| 实变函数     | Lebesgue 积分、测度论、函数空间基础            |
| 复变函数     | 复值函数、解析函数、留数、复积分               |
| 拓扑学基础   | 点集拓扑、连续性、紧致性、连通性等             |
| 泛函分析     | Banach、Hilbert 空间与算子理论                 |
| 抽象代数进阶 | 群论、表示论、域扩张、伽罗瓦理论（部分学校教） |
| 微分几何     | 流形、曲率、黎曼几何初步                       |
