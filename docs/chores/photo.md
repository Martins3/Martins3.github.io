速控转盘 1 和速控转盘 2 ，这台相机有两个转盘，分工如下：

速控转盘 1 (图中编号 16，位于机身背部的大轮子)：
- 这是主转盘，通常带有 SET 按钮。
- 它的功能最全，既可以调参数，也可以当方向键用（上下左右按压），还可以直接按下去确认。
- 在 M 档（手动模式）下，通常默认用它来调整光圈（也可以自定义）。

速控转盘 2 (图中编号 6，位于机身顶部靠后的拨轮)：
- 这是一个辅助转盘。
- 在 M 档（手动模式）下，通常默认用它来调整快门速度。
- 它的存在是为了让你双手握持相机时，右手食指和拇指可以分别控制两个参数，无需移动手指就能完成曝光调整，非常适合抓拍。

**三个**拨轮，这是为了实现“三指操作”，让你不用进菜单就能调整所有核心参数。默认分工如下：
1.  **主拨盘（快门前方，食指操作）：** 默认控制 **快门速度**。
2.  **速控转盘 1（机身背部大轮，图中编号16，拇指操作）：** 默认控制 **光圈大小**（F值）。
3.  **速控转盘 2（机身顶部后方，图中编号6，中指/食指操作）：** 默认控制 **ISO 感光度**。

1. **先明确题材**：
   - 人像 → 85mm f/1.4 或 135mm f/1.8
   - 风光/建筑 → 16-35mm f/4（广角+小光圈够用）
   - 旅行/日常 → 24-105mm f/4（恒定光圈变焦）
   - 野生动物 → 100-400mm f/4.5-5.6 + 防抖
2. **参数≠体验**：高规格镜头未必适合你。例如：
   - 85mm f/1.2 锐度可能不如 f/1.8，但氛围感更强
   - 变焦便利性有时胜过定焦画质
3. **二手/副厂考量**：
   - 副厂（Sigma、Tamron、Tokina）性价比高，注意固件兼容性
   - 二手镜头重点检查：镜片霉/划痕、对焦马达异响、光圈叶片油污
4. **系统生态**：一旦选定卡口，后续镜头投资具有沉没成本，换系统代价高。

**卡口/画幅匹配是前提，光圈/焦距定用途，锐度/防抖看体验，重量/价格衡取舍。** 建议先租后买，实际体验再决定。

## EOS R6 的各个模式的含义
<!-- 5bdbe995-4343-479c-a4c5-5c982d607390 -->

但其中有一个佳能微单专有的 **Fv 模式** 是很多新手容易忽略的“黑科技”。

首先记住
A : Aperture Value 光圈
T : Time Value

1.  **A+ (场景智能自动)：**
2. 创意拍摄模式：摄影核心 (P/Tv/Av/M) 这是大多数摄影师最常用的区域：
	* **P (程序自动)：** 相机帮你定光圈和快门，但你可以自己改 ISO、白平衡等。比 A+ 自由一点。
	* **Tv (快门优先)：** 你控制**快门速度**，相机自动配对光圈。
		* **建议场景：** 拍**运动物体**（如赛车、飞鸟用 1/2000s）或**丝滑流水**（用 1/2s 慢门）。
	* **Av (光圈优先)：** 你控制**光圈大小**，相机自动配对快门。**这是最常用的模式。**
		* **建议场景：** 想要**背景虚化**（大光圈 f/1.8）或**全景清晰**（小光圈 f/8-f/11）。
	* **M (手动曝光)：** 光圈、快门、ISO 全部手动调节。
		* **建议场景：** 光线极其稳定（影棚）或极其特殊（拍星空、烟花）时。
3. 佳能 R 系列特色：Fv 模式 (灵活优先 AE) 🌟
	* **用途：** 它是 P、Av、Tv、M 四个模式的合体。
	* **逻辑：** 在这个模式下，你可以随时手动控制光圈、快门、感光度或曝光补偿中的**任何一项或几项**，而剩下的项目会自动保持 Auto。
	* **黑科技之处：** 你不需要转动大拨盘切换模式，只需要通过拨轮就能在“全自动”和“全手动”之间无缝切换。
	* **操作小技巧：** 按 R6 上的“垃圾桶”键可以一键恢复全自动设置。

4. **B (B门)：** 手按多久，快门就开多久。
	* **建议场景：** 长时间曝光，如拍摄星轨、车流轨迹。
5. **C1, C2, C3 (自定义模式)：**
	* **用途：** 相当于“预设存档”。你可以把一套复杂的参数（比如拍鸟的高速连拍设置）存进 C1。
	* **建议场景：** 瞬间切换题材时。比如正在拍人像（C1），突然天上飞过一只鹰，瞬间转到 C2（拍鸟预设）。
6. 视频模式
	* **摄像机图标：** 切换到此模式后，相机会开启视频专用的菜单和界面。在视频下也可以再次选择 P/Av/Tv/M 曝光逻辑。

R6 快速选择建议
* **拍娃/拍宠/街头抓拍：** 首选 **Av 模式**，并将 ISO 设为自动。
* **拍风景/旅行：** **Av 模式**，光圈设在 f/8 左右。
* **想进阶练技术：** 尝试 **Fv 模式**，练习在不离开取景器的情况下调整各项参数。

在佳能（Canon）相机中，**Fv** 是 **Flexible-priority AE** 的缩写，中文翻译为“**灵活优先自动曝光模式**”。

### 1. Fv 模式核心：四个参数的自由组合

在 Fv 模式下，你可以对以下四个参数分别进行“自动（AUTO）”或“手动”设置，而无需拨动模式转盘去切换 Av、Tv 或 M 档：

* **光圈** (Aperture)
* **快门速度** (Shutter Speed)
* **ISO 感光度**
* **曝光补偿** (Exposure Compensation)

### 2. 它像是一个“任意门”

Fv 模式的神奇之处在于，它通过对这四个参数的控制，可以瞬间变成你熟悉的任何模式：

| 如果你设置...                       | 它就变成了...        |
| ---                                 | ---                  |
| 四个参数全部选 **AUTO**             | **P 档** (程序自动)  |
| 手动设定 **光圈**，其余 AUTO        | **Av 档** (光圈优先) |
| 手动设定 **快门**，其余 AUTO        | **Tv 档** (快门优先) |
| 手动设定 **光圈+快门**，ISO 为 AUTO | **M 档 + 自动 ISO**  |
| 手动设定 **光圈+快门+ISO**          | **M 档** (全手动)    |

### 3. Fv 模式的最大优势

* **切换飞快：** 以前你想从光圈优先（Av）换到全手动（M），得转动拨盘。在 Fv 模式下，你只需要拨动拨轮修改快门数值，它就自动从 Av 变成了 M，不需要眼睛离开取景器。
* **一键重置（归位）：** 这是 Fv 模式最强悍的地方。在佳能机身上，通常按下**垃圾桶图标（删除键）**，就能瞬间把选中的参数变回 AUTO；长按则能让所有参数全部回到 AUTO。
* *场景示例：* 你正在手动调参数抓拍鸟类，突然光线变了手忙脚乱，按一下删除键，相机立刻接管快门和 ISO，保证你先能拍到画面，不会废片。


### 4. 操作小技巧

* **带下划线的数值：** 在 Fv 模式下，如果数值下方有下划线，代表它当前处于 **AUTO** 状态。
* **橙色转盘图标：** 屏幕上参数旁边会出现一个橙色的转盘小图标，代表你当前拨轮正在控制哪一个参数。

### 总结

**Fv 模式 = P + Av + Tv + M 的集成体。** 它非常适合那些想在拍摄过程中快速切换控制权、但又不想在转盘上转来转去的摄影师。

## 相机光圈
<!-- 5ccaa6bc-700f-425e-8e55-87b2c27830f0 -->

这是一个非常棒的深度问题。很多摄影爱好者知道“光圈大虚化好”，但从光学物理层面来看，这本质上是一个**几何光学中的“光束锥角”与“弥散圆”的问题**。

我们可以通过以下三个步骤，从光学原理上拆解这个现象：

### 1. 唯一的光学焦点与“弥散圆” (Circle of Confusion)

在理想的光学系统中，对于给定的对焦距离，空间中只有**一个平面**（焦平面）上的点能汇聚成传感器上的一个**完美的点**。

* **焦平面上的点：** 经过镜头折射后，光线精准汇聚在传感器上。
* **焦平面以外的点：** 无论是在焦点前还是焦点后，它们发出的光线在穿过传感器时，还没来得及汇聚（焦点在传感器后）或者已经汇聚并开始扩散（焦点在传感器前）。
* **结果：** 这些不处于焦点上的点，在传感器上呈现出的不是一个“点”，而是一个**小圆斑**。光学上称之为**弥散圆 (Circle of Confusion)**。

当这个圆斑足够小，小到人眼分辨不出来它是个圆，而认为它还是个“点”时，我们就认为它是“清晰”的。这段视觉上清晰的距离范围，就是**景深**。

### 2. 核心光学分析：光束的“锥角”

为什么改变光圈大小会影响这个圆斑的大小？请看下面的几何关系：

* **大光圈 (如 f/1.8)：** 镜头开口大，参与成像的光束是一个“**粗壮的圆锥**”。由于圆锥的边际角（收敛角）很大，光线在离开焦平面后会**迅速扩散**。这意味着稍微偏离一点焦点，弥散圆就会迅速变大，超过人眼的辨识极限。所以，大光圈的清晰范围非常狭窄（浅景深）。
* **小光圈 (如 f/16)：** 物理孔径被遮挡，只有靠近镜头中心的光线通过，形成的是一个“**细长的圆锥**”。由于这个圆锥非常尖细，光线在离开焦平面后的**扩散速度非常慢**。即使点稍微远离了焦平面，它在传感器上形成的弥散圆依然很小，保持在人眼认为“清晰”的范围内。所以，小光圈的清晰范围非常广（深景深）。

### 4. 极端情况的副作用：衍射 (Diffraction)

既然光圈越小景深越深，那为什么我们不一直用 f/32 拍风景呢？

这里涉及另一个物理光学现象：**衍射**。当光圈孔径小到一定程度，光波在经过孔隙边缘时会产生明显的衍射（波的叠加），导致原本应该很清晰的“细长圆锥”末端变得模糊。

* 在你的 **佳能 R6** 这种高画质机身上，通常光圈小于 **f/11** 或 **f/16** 后，你会发现画面整体锐度开始下降，这就是光学衍射限制了画质。

### 总结

* **大光圈 = 宽光锥 = 弥散圆扩张快 = 景深浅**
* **小光圈 = 窄光锥 = 弥散圆扩张慢 = 景深深**

## 相机焦距
<!-- 22374100-9ca3-4358-9ae4-2035f0cebc01 -->
在摄影镜头参数中，**24-105** 代表的是这款镜头的**焦距范围**，单位是毫米（mm）。

简单来说，这两个数字告诉了你这个镜头的“视野范围”：它能拍多宽，又能拉多近。

### 1. 24 代表“广角端” (Wide-angle)

* **含义：** 这是镜头的最小焦距。
* **视觉效果：** 数字越小，视野（视角）就越宽。
* **用途：** 24mm 属于**广角**，非常适合拍摄宏大的场景。比如拍摄壮丽的风景、高大的建筑，或者在狭窄的室内拍下全景。

### 2. 105 代表“长焦端” (Telephoto)

* **含义：** 这是镜头的最大焦距。
* **视觉效果：** 数字越大，視野就越窄，但能把远处的物体“拉近”放大。
* **用途：** 105mm 属于**中长焦**，适合拍摄人像或特写。它可以产生很好的“背景虚化”效果，让主体更加突出，同时也适合拍摄距离稍远的景物。

### 3. 24-105 组合在一起意味着什么？

这是一款**变焦镜头**（Zoom Lens）。这意味着你不需要更换镜头，只需拧动镜头上的变焦环，就可以在 24mm 到 105mm 之间自由切换。

* **全能选手：** 这个焦段覆盖了从“广角”到“标准”再到“中长焦”的常用范围。
