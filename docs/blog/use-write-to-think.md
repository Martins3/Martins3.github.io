# 用写作来重新思考问题

我写博客存在多种动机:
1. 很多小细节记录了，之后免得重新再次 Google 或者问 ChatGPT ，类似这个
	- https://github.com/jbranchaud/til
	- https://github.com/chenzhiwei/linux
2. 总体来说，中文社区的质量比英文的差很多，而且 CSDN 之类的毒瘤，鼓励抄袭盗版，搞中文社区更加是乌烟瘴气。改善这种情况，我认为每一个都从自己做起，尽量输出一些自己的原创内容。

3. 我高中的时候，每次语文作文都打了一个很低的分数。
我问我的语文老师: 为什么我们非要浪费时间写一些毫无意义的东西，这些作文既没有表达什么思想，也没有高效的记录内容。
语文老师反问我: 那你有你觉得比较好的形式的作文吗，可以拿出来给我看一下。

当时我不知道如何作答，那么现在我可以说，这个仓库就是我对于"作文"的理解，不要废话，迎合，形式，神秘主义，800 字，只可意会，只有记录，分析和表达。

但是我想说的，最关键的作用是，在写的过程中，我在重新思考问题:
记录笔记是输入的过程，写笔记是输出的过程，在写作的过程中，可以重新回忆，从不同的角度理解问题，全面梳理问题，可以加深对于问题的理解。
博客是一个人构建知识体系的过程。
写博客应该是系统的，深入的，而不是零散的，博客不是笔记，而是思考过程的体现。
在 博客中应该存在互联网中之前没有的东西，不是告诉这个东西是什么，而是反映了你对于这件事情的思考和创新。

[Write about what you learn. It pushes you to understand topics better.](https://news.ycombinator.com/item?id=37118883)
