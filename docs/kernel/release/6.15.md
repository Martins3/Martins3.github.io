## 第一部分 : https://lwn.net/Articles/1015414/
- The XFS filesystem now supports zoned storage devices.

- Work continues toward the breaking up of the RTNL lock (the "big networking lock"), which is a contention bottleneck in much of the networking subsystem.

- Initial support for zero-copy data reception via io_uring has been added.

## 第二部分 : https://lwn.net/Articles/1016119/

- It is also now possible to read epoll events via an io_uring operation.

- The runtime verification subsystem has a new feature called "scheduler specification monitors" that allows multiple monitors to run concurrently and interact with each other.
