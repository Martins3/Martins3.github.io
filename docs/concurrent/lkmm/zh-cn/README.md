# LKMM 文档中文辅助材料

这个目录为 `tools/memory-model/Documentation/` 下的每个源文件提供两份辅助材料：

- `*.translation.md`：面向理解的结构化中文译解，尽量保留 LKMM 术语。
- `*.summary.md`：快速阅读版总结，突出结论、适用场景与阅读建议。

说明：

- 这些文件的目标是帮助阅读原始 LKMM 文档，不是替代原文。
- 对超长文件，尤其是 `explanation.txt`，这里采用章节级译解而不是逐句直译。
- 术语如 `po`、`rf`、`co`、`hb`、`pb`、`rfe`、`READ_ONCE()`、`smp_store_release()` 等保持原名，避免与内核代码脱节。
