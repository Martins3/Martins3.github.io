## rds
- [What are RDS or reliable datagram sockets](https://stackoverflow.com/questions/27980309/what-are-rds-or-reliable-datagram-sockets) 对应 net/rds 目录

似乎不是有很多人用，也差不多是这个意思:

> [!NOTE]
> 参考神奇海螺的意见，有待验证

  RDS is not deprecated but it serves a very specific niche - high-performance computing environments that leverage InfiniBand networks,
  primarily within Oracle's ecosystem and some enterprise database deployments. While it receives ongoing maintenance and security fixes,
   it's not seeing major new feature development. The technology remains relevant for its intended use case but has a limited user base
  compared to mainstream networking protocols.
