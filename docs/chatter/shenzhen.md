# 深圳

和北京比起来，似乎还不错，以至于我可以心平气和的写一些见闻。


1. 公园多，山地也多，导致交通受阻，可用面积少。
2. 冬天短（2025-10-13 的时候还是短袖）
3. 租房稍微便宜，距离科兴科学园 5 分钟步行的，30 平，合租，大约 4000
4. 绿植覆盖率高
5. 早餐，最常见的就是肠粉

有趣的体验，可以顺着沙河，一路骑车到海边。

- 塘朗山
- 西丽湖

## 南山博物馆

不得不感触，有钱真好，既然南山在改革开放之前只是一个小渔村，
要建一个博物馆咋办呢?

- 可以和其他有资源的博物馆联名，
	- 例如敦煌，鄂尔多斯
	- 穆夏
- 挖掘一下县志
- 讲讲近代史

我停留时间最长的地方居然是一楼的商店里面的冰箱贴，
纠结了很长时间要不要买一个，最后放弃了，有点贵。
