# 看似有关系，实际上没关系

记录一些名字、发音或外形相似，但实际上毫无关联的事物。

## 技术圈

- **LLVM / vLLM**
  - LLVM：编译器基础设施项目（Low Level Virtual Machine）
  - vLLM：大语言模型推理和服务引擎
  - 关系：拼写像镜像，实际八竿子打不着

- **Java / JavaScript**
  - Java：Sun 公司推出的面向对象编程语言
  - JavaScript：网景公司搞的脚本语言，为了蹭 Java 热度强行改名
  - 关系：名字像父子，实际是陌生人

- **KVM（虚拟化）/ KVM（切换器）**
  - KVM（Kernel-based Virtual Machine）：Linux 内核的虚拟化模块
  - KVM（Keyboard, Video, Mouse）：机房用来切换多台电脑的外设共享设备

不过，我这里需要说下，其实绿联有自己一个 kvm 切换器

## 地理与国际

- **卡巴斯基 / 巴基斯坦**
  - 卡巴斯基（Kaspersky）：俄罗斯网络安全公司
  - 巴基斯坦（Pakistan）：南亚国家
  - 关系：中文读起来节奏像，实际毫无关联

- **澳大利亚 / 奥地利**
  - Australia：大洋洲国家
  - Austria：中欧国家
  - 关系：英文名字像，中文不像，但很多人搞混

- **摩纳哥 / 摩洛哥**
  - 摩纳哥（Monaco）：欧洲城邦国家，靠地中海
  - 摩洛哥（Morocco）：北非国家
  - 关系：中文名字像，地理位置天差地别

- **斯洛伐克 / 斯洛文尼亚**
  - 斯洛伐克（Slovakia）：中欧内陆国
  - 斯洛文尼亚（Slovenia）：中欧南部的沿海小国
  - 关系：名字前缀一样，建国时间也差不多，容易搞混

## 日常与其他

- **奇异果 / 奇异博士**
  - 奇异果：猕猴桃的别称（Kiwi fruit）
  - 奇异博士：漫威超级英雄 Doctor Strange 的中文译名
  - 关系：都有"奇异"二字，实际风马牛不相及
