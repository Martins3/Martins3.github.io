# 远程工作体验

先说结论，至少不适合我。

## 开始的仓促
1. nixos 上始终没有解决 vpn 的问题，只能通过另外的一个 Windows 电脑访问。
2. 没有带降噪耳机，键盘实在是太吵了。
3. 很多仓库都没有同步到本地的拯救者笔记本上，还是需要远程登录到公司的机器上操作，有点笨拙。

## 长期稳定测试
- 好处
  - 小区环境比北京好很多，离开小区就是河畔公园，因为我总是在用眼过度的边缘，所以我很喜欢绿树成荫，生机勃勃的感觉。
  - 可以直接放外音，考虑到现在只是使用的电脑内置的音响，如果可以长期居家，买一个大音响，效果惊人。

- 坏处
  - 每次开机都需要连 vpn ，没有没有充会员，电脑 suspend 之后，又需要连一次。在 Mac 和 Windows 上还好，在 Nixos 上真的有点琐碎了。
  - 无法逃避的孤独感。 在公司上班的时候，最有趣的部分在于中午和晚上吃饭的时候和同事聊聊天。
居家的时候，这部分没有了。我在我所在的城市，没有朋友。

## 感触

我以为居家办公就会工作的轻松点，实际上并不会，基本上没有任何区别。

首先，我有自己的工作在做，而且几乎每过几十分钟就会有人来找我，
最后，关于 Linux 内核，我记录成千上万的问题，它们都需要被整理。
最后的结果就是，我几乎永远在电脑前，这和我上班的时候没有区别。
