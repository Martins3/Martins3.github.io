# 为什么我不用小红书

信息茧房，互联网奸臣，让人逃避现实

使用小红书过多的问题:
1. 你看别人的父母多好，我被原生家庭拖累。
2. 你看别人的男朋友/女朋友多好，我的一点都不浪漫/体贴。
3. 教人如何短视
  - 考公务员好，但是公务员的工资从哪里来不需要思考?

但是成年人的想法是很难改变的，小红书只是放大了人的懒惰，

小红书的根本问题是，他让我们的生活变的过于多姿多彩，
这里的多姿多彩不是一个贬义词，
没有小红书，我们不会发现原来一直被埋没的美景，美食。

但是，如果有空闲时间就打开小红书看看，人的无聊时间都被消耗了。

## 小红书的真正问题是他消耗了我们的无聊时间

无聊不是一个坏事情

无聊之后，我们才可以打扫卫生，养花，

可以体会一下，人类持续奋斗几千年，终于得到了闲暇，不应该被

## 为什么我们会沉迷于小红书

- 小红书的运营 制作的归属感，忽然，所有的人都在玩同一个梗

## 小红书特色

1. 个人成长 : 不就是成功学吗？更烦的是，他还想靠这个挣我的钱。

## 不用小红书，我可以做什么才是关键

1. 听博客
2. 定期反思生活中的各种问题
3. 睡觉
4. 学习新的技能


社交媒体让人极度焦虑，而且还发现自己做不了什么?
https://www.ruanyifeng.com/blog/2026/01/weekly-issue-380.html

> （4）社交媒体则使得人们永远不会对现状满意。
>
> 以前，你的参照群体只是周围人群，现在的参照群体是全世界。你每天看到的都是收入高、赚钱容易、生活优渥的人群，永远会让你感到自己的生活不够好，而无论你已经取得了怎样的成就。

我的嘴替了:
https://www.noahpinion.blog/p/trapped-in-the-hell-of-social-comparison?hide_intro_popup=true

可能的对策包括：
- 提供更多公共品：通过建设优美的公园、步行街、免费沙滩等，让生活质量的差距不完全依赖于私人消费。
	- 如果，你想要个性化的消费，其代价极大
		- 从普通人的角度来说，该如何解决? 主动识别那些，强调有多难获取，有多么贵，有多么私人，而不是这个东西有多好吃，好玩的项目，都是需要直接否决的
- 文化引导：培养类似日本那种“富人不在公共场合过度炫富”的文化。
- 数字素养：引导年轻人减少使用 TikTok 和 Instagram，并意识到网红展示的生活并非现实的衡量标准。

终于知道了， 飓多多这种脑瘫节目 让我感觉到不悦了。
