## 饥荒 don't starve
<!-- 2acfaf61-d94d-4383-8833-159438976fb9 -->

https://www.bilibili.com/video/BV1n8411d7SL
还是那个问题，可以文字整理一下吗?

https://zhuanlan.zhihu.com/p/656374033 :
1. 黑夜来临时，可以用火把点燃身边的树，节省火把耐久，开局秋天，夜晚较短，晚上烧一颗树就可以度过一个夜晚。注意！不要烧靠在一起的森林，有可能烧掉一整片森林，只需要寻找落单的树即可。
2. 在保证自己手里有一定的草和树枝后，便可以开始探险。游戏开局，玩家会没有目标，首先要做的就是探图，在游戏初期将地图全部跑亮，有助于建家的时候到对应地点收集资源，同时也能将许多重要的地点探索出来以便于后边来去方便
3. 在探索地图的时候，可以看到地面上有花朵，空手就可以捡起来，每一朵花可以回复5点的精神值，捡到的花在制作栏中可以合成花环，装备花环后可以持续回精神值。
4. 在黄昏时间段，有可能在地面上看到绿蘑菇，绿蘑菇只在黄昏会露出地面，将绿蘑菇烤熟食用回复15精神值，注意一定要烤熟！饥荒中多数食物在烤熟后有回复加成，烤熟需要制作火堆，将火堆放于地面，鼠标点起食物放到火堆上便会显示烹饪，鼠标左键便可以烤熟。
5. 饥荒中食物有新鲜度特性，颜色由绿色，黄色，红色分别表示新鲜，不新鲜，快坏了。
6. 白天有花的地方会刷新，蝴蝶，可以击杀蝴蝶食用，一只蝴蝶可以回复8滴血。（蝴蝶打法：移动人物将人物影子与蝴蝶影子相重合，按下ctrl键加f键强制攻击，即可击杀蝴蝶，强制攻击操作可以攻击一些中立生物。顺提一句，键盘F为攻击键，空格键为拾取物品，采集等。）前期还可以打蜘蛛掉蜘蛛腺体回血，萌新不推荐直接打怪。前期先怂一怂，不然容易死亡，死亡次数多了，容易影响游戏体验，稳才是硬道理。

## games 游戏给到我的启示
<!-- 6d853e33-c5d6-403b-8a4f-59ec5d19d2a4 -->

1. 不要重新发明轮子，先看教程，大概率没问题
2. 先看教程，然后再看动手
	- 然后理清楚所有的细节，做了之后，再去测试，记录其中那些东西是不符合预期的。
	- 这里有一个问题在于，教程记不住，但是实际上统计下来的，背教程的时间也会远远小于反复测试的时间
4. 不看教程就去玩，可能会比较痛苦，不过这里可能需要区分一下，到底是思考痛苦(短时间接受太多信息)，还是认知方式带来的，还是教程并不好
	- 教程一下子看的东西太多了，但是打一局游戏，才学一个英雄
	- 教程是文字，但是游戏内部是图形
	- 教程是面面俱到，但是游戏是最核心的东西
3. 一次性击穿一个，永远都是最方便的，不要自己骗自己
