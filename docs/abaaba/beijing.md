## 更多的思考
很多人做事情不是为了目的，而是为了达到目标，例如玩原神的时候，
你总是会忍不住去刷各种材料，最后都忘记了为什么要玩这个游戏。

类似的，很多人做事，都是为了表达一个想法，例如这样做，证明我很厉害。
例如，在北京呆着，说明我就有一种，来到了一个更好的平台。

这个有点像做 Linux 一样，很多人沉迷于刷 kernel patch ，而不是想着如何解决内核中的各种问题，我认为这样是本末导致了。
