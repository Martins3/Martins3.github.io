# 关于读博的思考

前段时间看了[阮一峰的周刊](https://www.ruanyifeng.com/blog/2024/07/weekly-issue-308.html)，里面讨论"工作找不到，博士能读吗"这个问题，引发了我的一些想法，在此记录一下。

我读硕士时是极力反对读博的，因为我观察到很多人读博是为了逃避工作，而不是为了解决真正的问题。
最近找工作时浏览招聘网站，发现博士的待遇和薪酬是相当有竞争力的。
但是我总体的想法是没有变化的

做任何事情都是为了解决问题，而不是为了逃避问题。如果你读博士是发现可以解决什么问题，那我认为你应该读。如果你只是为了晚一点去工作，那你这一辈子都会很痛苦
