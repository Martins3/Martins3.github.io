# 不要辜负这个伟大的时代

我的童年的早期记忆中，就是我母亲教我背诗，这首诗是 锄禾日当午，。。。

当时我不太理解。

等我上小学的时候，在农村里参与了一下农业生产，我才意识到，所以在乎粮食是非常重要的
这种艰苦的生活没有持续太久，我就县城里面念小学了。

等我上了高中，加深了物理的理解，意识到植物需要的氮磷钾，
意识到人类已经可以从空气中捕获氮肥的时候，

锄禾日当午 我有全新的理解，当你无法获取氮肥，你无法使用机械自动化的时候，
那么就必须告诉粮食很重要。

现在，我们可以通过显卡，
甚至我们真的需要想想，当物质无限供应，当人类从繁重简单的劳动中解放出来的时候，
我们终于有时间来解决哪些长期困扰人的问题
